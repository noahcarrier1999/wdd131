Overview

Purpose

The Purpose of this Web application is to allow users to track their expenses and make it easier for them to budget. It will allow them to see where they spend money and plan a budget according to their expenses

Audience

The audience for this application is anyone who is wanting to keep track of their expenses

Dynamic elements

This website will be dynamic by adding and displaying the transactions in real time it also will use the reduce() method to help calculate the total balace and it will use the filter() method to allow the user to filter through the transactions. Additionally, we will store the data in localStorage
